# `@mavrykdynamics/mavlet-transport-matrix`

This package is part of the `@mavrykdynamics/mavlet-sdk` project. [Read more](https://github.com/mavryk-network/mavlet-sdk)

## Introduction

This package contains methods to facilitate communication over the Mavlet network, a decentralised P2P network that is based on the matrix protocol.

Unless you need to import types from this package, you probably don't need to add it to your project.

Check our documentation for more information. [Documentation](https://docs.mavlet.mavryk.org)
